Welcome to eTuneBook


With eTuneBook you can

  • Manage Abc tunes in your browser.
  • Edit tunes by changing Abc text (Tune-Editor).
  • Edit single notes by choosing from note-element options (Note-Editor).
  • Build tune sets.
  • Specify repeat time or add a comment to set positions.
  • Build playlists from tune sets.
  • Specify playlist parameters like name (e.g. venue of a gig), event (e.g. Ceili) and band.
  • Specify when during a playlist a tune set is played (position) and add an alternative tune set name or a comment specific for the playlist position.
  • Override Repeat and Annotations of a Tune in a Playlist.
  • Specify Part Play Infos for a Tune in a Playlist.
  • View single sets showing the first few bars of each tune.
  • Remember when you last played a tune and how good you were.
  • Label tunes with a color or add a comment.
  • Transpose tunes.
  • Store links to other sites holding tune informations.
  • Store links to YouTube videos and view them in eTuneBook directly.
  • Easily identify those tunes you should rehearse. Just sort by last modification date, last play date, rehearse frequency or skill.
  • Random choose tunes and sets within given play ranges.
  • Search tunes and sets by title.
  • Filter tunes and sets by type, key, color, skill, event, band, play date, modification date, frequency.
  • Import Abc tunes from a File on your hard disk or from your Google Drive.
  • Export Abc tunes to a File on your hard disk or to your Google Drive.

Tested /Recommended combinations of browsers and environments:

  • Chrome / Windows7
  • Firefox / Windows7
  • Chrome / iOS7 (iPhone, iPad)
  • Safari / iOS7 (iPhone, iPad)

Import/Export via local hard disk does not work on mobile iOS devices. You have to use Google Drive instead.
